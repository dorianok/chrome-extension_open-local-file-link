Since Google Chrome links to local files with the prefix file:// are disabled, 
this extension intercepts click events on links with local file links to open them in a new tab.

[Security]
This extension only reacts to trusted click events.

[Others]
There is no personal information processed or used in this extension.
The source code is available at GitHub (see "Website" link). It is a fork of Takashi Sugimoto's Enable local file links extension with additional support for SVG a tags.
